# Air-Quality-Open-Data
Ce projet vise à essayer de pratiquer les données publiques ouvertes sur la qualité de l'air dans les régions Françaises. L'architecture pensée est  : Postgresql pour le schema relationnel , Python pour la collecte via API et traitement en temps réel. Le but est d'expérimenter l'ingestion des données pour une base pereine et contrôlée.
